# TimeFlow · 时间流

> 一款专注于「时间感知」的微信小程序，让过去有迹可循，让未来有所期待。

---

## 功能概览

### 主界面 · 倒计时
- **未来 / 过去** 双 Tab 切换，分别展示倒计时与已过去时长
- 滑动数字翻牌时钟动画（未来事件向下翻，过去事件向上翻）
- 事件置顶功能，支持手动选择当前关注事件
- 未来事件倒计时归零时，展示「未来已来」动画覆盖层
- 系统内置事件：今天下班、本周末（仅工作日显示下班倒计时）

### 日历页
- 月视图日历，支持手势左右滑动切换月份
- 标记法定节假日（红）、传统节日（蓝）、国际节日（紫）、自定义事件（青）
- 点击日期弹出当天事件详情
- 月度事件分类列表：法定节假日 / 传统节日 / 国际节日 / 我的事件

### 个人页
- 微信头像 & 昵称（支持编辑）
- 事件分类展示：
  - **进行中**：未来未到期事件 + 过去未超 1 年的事件
  - **已归档**：未来已到期事件（子列：未来事件 / 循环事件）+ 过去超 1 年事件（子列：过去事件）
- 左滑事件卡片可编辑 / 删除

### 新建 / 编辑事件
- 事件类型：一次性事件 / 重复事件（过去日期仅支持一次性）
- 目标日期 + 目标时间选择器
- 循环规则：每年 / 每月 / 每周
- 备注字段
- 创建规则校验：
  - 不能创建起始时间距今超过 1 年的事件
  - 过去超 1 年的事件自动归档并从主界面移除

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | 微信小程序原生（WXML / WXSS / JS） |
| 后端 / 数据库 | 微信云开发（Cloud DB + Cloud Functions） |
| 云函数 | login · getHolidays · syncEvents · updateHolidays · refreshRecurring |
| 农历计算 | 内置农历工具库（utils/lunar.js） |
| 节假日数据 | 内置 2026 年法定节假日 + 国际节日，支持云端更新 |

## 核心设计理念

- **过去有迹可循**：支持记录历史时间点，查看已过去多久
- **未来有所期待**：支持倒计时事件，感知时间流逝
- **边界保护**：不能创建距今超过 1 年的事件，超期自动归档
- **克制的体验**：进行中最多 20 个未来事件 / 10 个过去事件

## 本地开发

1. 克隆仓库
   ```bash
   git clone https://github.com/JettW-g/time-flow.git
   ```
2. 使用 **微信开发者工具** 打开项目根目录
3. 在 `project.config.json` 中替换 `appid` 为你自己的小程序 AppID
4. 开通微信云开发，创建对应集合：`events`、`holidays`、`users`
5. 部署云函数（右键云函数目录 → 上传并部署）
